Create an S3 bucket.

1) Create an S3 bucket for the CF template.
	aws s3 mb s3://myname-cf-demo

2) Push the CF template to the S3 bucket.
	aws s3 cp lamp.cf.template.json s3://myname-cf-demo

3) Create the CF stack.
	aws cloudformation create-stack --stack-name my-cf-demo --template-body s3://myname-cf-demo/lamp.cf.template.json
